>> 876 破裂(haretsu) = explosion, rupture
per esempio la foratura di un ruota / >> puncture = タイヤの破裂(taiya no haretsu)
>> 875 列 -> 行列(gyouretsu) oltre al significato di matrice, nel linguaggio comune viene utilizzato per indicare la fila di persone per esempio davanti ad una panetteria.
>> 874 殖 -> 増 殖(zoushoku) = increase , utilizzato per esempio in contesto chimico e medico per indicare l'incremento, la diffusione di una sostanza, come per esempio l'incremento di un batterio.
>> 870 被害(higai) = damage,injury, harm
note: questo kanji e' usato anche in 被告(hikoku) = coupable (accusato, nell'ambito di un processo), il cui contrario e' 原告(genkoku = l'accusatore generatore 原 del information 告)
>> 869 破る(yaburu) = to tear, to break, to destroy
usato per esempio anche in espressioni pagina strappata = 破れた紙(yabureta kami)
>> 868 披露(hirou) = announcement, unveiling
ritrovi questa kanji anche in 披露宴(hirouen) = wedding party.
